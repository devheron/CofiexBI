📋 Sobre este projeto

Este dashboard apresenta uma análise profunda dos financiamentos externos gerenciados pela COFIEX (Comissão para Assuntos Financeiros Externos). Os dados foram submetidos a um rigoroso processo de limpeza, validação e transformação, preservando a integridade dos registros originais enquanto criava indicadores derivados para apoiar análises comparativas e detecção de padrões. O projeto responde a dez questões chave sobre concentração territorial, participação de fontes financiadoras, composição da carteira e mudanças históricas.
